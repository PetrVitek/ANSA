Detailed Description

A C++ interface to abstract the functionality of IRoutingTable.

Referring to IRoutingTable via this interface makes it possible to transparently replace IRoutingTable with a different implementation, without any change to the base INET.

See also:
IRoutingTable, IPv4Route
