Detailed Description

Stores an IPv4 or an IPv6 address.

This class should be used everywhere in transport layer and up, to guarantee IPv4/IPv6 transparency.

Storage is efficient: an object occupies size of an IPv6 address (128 bits=16 bytes) plus a bool.
