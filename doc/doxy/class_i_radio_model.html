Detailed Description

Abstract class to encapsulate the calculation of received power of a radio transmission.

The calculation may include the effects of path loss, antenna gain, etc.

virtual double IRadioModel::calculateDuration ( AirFrame *  ) [pure virtual]

Should be defined to calculate the duration of the AirFrame.

Usually the duration is just the frame length divided by the bitrate. However, in some cases, notably IEEE 802.11, the header has a different modulation (and thus a different bitrate) than the rest of the message.
