Detailed Description

In this file all rtcp packet types are declared: There is a the superclass RTCPPacket which is not intended to be used directly.

It defines an enumeration to distinguish the different rtcp packet types and also includes header fields common to all types of rtcp packets. Direct subclasses are RTCPReceiverReportPacket, RTCPSDESPacket and RTCPByePacket. RTCPSenderReportPacket is declared as a subclass of RTCPReceiverReportPacket because it only extends it with a sender report. Application specific rtcp packets are not defined. The class RTCPCompoundPacket isn't derived from RTCPPacket because it just acts as a container for rtcp packets. Only rtcp compound packets are sent over the network.
